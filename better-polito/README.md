# Polito Community

> **⚠️ UNOFFICIAL**: This is a community-built, unofficial student portal. It is **not affiliated with, endorsed by, or connected to Politecnico di Torino**.
>
> Based on [polito/students-app](https://github.com/polito/students-app) — licensed under EUPL v1.2.

An AI-enhanced web portal for PoliTO students, built as an unofficial community project.

## Features

- 📅 Agenda & calendar view
- 📚 Courses, files & materials
- 🎓 Transcript & grades
- 📝 Exam booking
- 🗓️ Room & service bookings
- 🗺️ Campus map
- 🎫 Support tickets
- 👥 People directory
- 📰 News & job offers
- 🤖 AI Study Planner
- 🤖 AI Chatbot over guides/services
- 🤖 Course material summarizer
- 🤖 Grade analytics & predictions

## Setup

```bash
cp .env.example .env.local
npm install
npm run dev
```
